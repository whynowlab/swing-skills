// 문제

당신의 AI는
생각을 잘못하고 있습니다.

틀린 답이 아닙니다. 틀린 과정입니다. 기본 상태의 AI에는 어떤 프롬프트로도 고칠 수 없는 인지 편향이 내장되어 있습니다.

⚡
첫 번째 답 편향

AI는 가능한 첫 번째 답을 고릅니다. 최선의 답이 아니라. 정확하기보다 도움되어 보이는 걸 최적화합니다.

🪞
동의 함정

반박해야 할 때 동의합니다. 당신의 아이디어가 좋다고 하면 — 틀려도 맞는 이유를 찾아냅니다.

🌫
리서치 연극

검증 없이 추측으로 리서치합니다. 자신감 넘치는 출력, 하지만 소스 검증이나 등급 분류는 없습니다.

// 7개 메타인지 스킬

완전한
스택.

각 스킬은 인지 모듈입니다. 하나만 설치해도 됩니다. 전부 설치해도 됩니다. 조합해서 쌓으세요.

01 / 07
cross-verified-research

4단계 리서치 파이프라인. 소스 등급(S/A/B/C) 부여, 독립 소스 간 교차 검증 후 출력.

리서치
02 / 07
creativity-sampler

전형성 편향에 맞서는 확률 가중치 5개 옵션. 참신성과 실행 가능성을 평가한 뒤 비관습적 대안 도출.

의사결정
03 / 07
adversarial-review

모든 결정에 대한 3벡터 악마의 변호인 공격. 근거 없이 통과 판정 불가.

리뷰
04 / 07
deep-dive-analyzer

5개 분석 렌즈로 복잡한 시스템을 현미경 해부. 가정, 숨겨진 의존성, 엣지 케이스까지.

분석
05 / 07
skill-composer

다중 스킬 파이프라인 오케스트레이션. 순차/병렬 워크플로우로 복합 인지 작업 처리.

메타
06 / 07
persona-architect

맥락 인식 AI 페르소나 설계 프레임워크. 추론 스타일, 제약 조건, 인지 전문화를 구조화.

메타
07 / 07
tiered-test-generator

다단계 난이도 지식 검증. L1 암기부터 L9 종합까지, 단순 Q&A로 발견 못하는 이해 공백 노출.
